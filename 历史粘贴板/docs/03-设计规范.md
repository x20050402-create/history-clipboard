# 03 · 设计规范（UI）

> 风格：简洁、直观、留白舒适。主色调淡蓝色。

## 1. 主题色板

| 用途 | 名称 | 色值 | 说明 |
| --- | --- | --- | --- |
| 主色 | primary | `#5B9BD5` | 按钮、强调、置顶标记 |
| 主色深 | primary-dark | `#4A86C0` | 悬停 / 按下 |
| 浅底 | bg | `#F4F8FC` | 窗口背景（淡蓝白） |
| 卡片底 | card | `#FFFFFF` | 卡片背景 |
| 置顶卡片底 | card-pinned | `#EAF3FB` | 置顶卡片淡蓝底 |
| 强调浅 | accent-soft | `#DCEBF8` | 搜索框聚焦、标签底 |
| 主文字 | text | `#2C3E50` | 正文 |
| 次文字 | text-muted | `#8A9AAB` | 时间、摘要、提示 |
| 分隔线 | border | `#E2EAF2` | 卡片描边、分隔 |
| 危险 | danger | `#E2706E` | 删除按钮 |

## 2. 字体与排版

- 字体：系统默认（`Microsoft YaHei, "Segoe UI", sans-serif`）
- 字号：标题 15px / 正文 13px / 摘要·时间 12px
- 行高：1.5；卡片正文最多显示 3 行，超出省略号

## 3. 间距与圆角

- 窗口内边距：12px
- 卡片间距：10px；卡片内边距：12px
- 圆角：卡片 10px / 按钮 8px / 搜索框 8px
- 阴影：卡片 `0 1px 3px rgba(91,155,213,.12)`

## 4. 布局结构

```
┌─────────────────────────────────┐
│  🔍 搜索框 ……………………     ⚙设置  │  顶部栏（固定）
├─────────────────────────────────┤
│  ★ 置顶区（若有置顶卡片）          │
│  ┌───────────────────────────┐  │
│  │ 卡片：内容预览              │  │
│  │ 12:30 · 文字     📌 🗑      │  │
│  └───────────────────────────┘  │
│  普通区（时间倒序）               │  滚动区
│  …                              │
├─────────────────────────────────┤
│  ⓘ 软件运行时才会记录复制内容      │  底部提示
└─────────────────────────────────┘
```

## 5. 卡片三态

- **默认**：白底、浅描边
- **悬停**：阴影加深，右上角显示 📌 / 🗑 操作按钮
- **置顶**：淡蓝底 `#EAF3FB` + 左侧主色竖条 + 📌 高亮

## 6. 卡片内容样式

- 文字卡片：显示文字预览（最多 3 行）
- 图片卡片：显示缩略图 + "图片 宽×高"
- 文件卡片：📄 图标 + 文件名（多个用逗号分隔），可"在文件夹中显示"

## 7. 控件样式

- 搜索框：浅底、聚焦时描边变主色、左侧放大镜图标
- 按钮：主按钮主色填充白字；图标按钮透明底、悬停浅蓝底
- 时间显示：相对时间（刚刚 / 5分钟前 / 今天 12:30 / 昨天 / 日期）

## 8. 文案

- 空状态：「还没有记录哦～复制点什么试试吧」
- 未运行提示（底部）：「软件运行时才会自动记录，关闭后到下次打开之间复制的内容不会保存」
- 删除确认：单条删除可直接删（带轻微动画），不弹确认框（降低打扰）
