# 01 · 需求文档（PRD）

> 软件名：剪贴历史（clip-history） · 平台：Windows · 类型：本地历史剪贴板工具

## 1. 项目目标

Windows 自带剪贴板只保留最近一次复制的内容。本软件在后台自动记录最近一段时间内复制过的**文字、图片、文件**，按时间倒序展示成卡片，方便随时回看、搜索、重新复制粘贴，并按天数 / 条数自动清理。

## 2. 功能需求

### 2.1 内容记录（软件运行时自动进行）
- 文字（纯文本）
- 图片（截图、复制的图片）
- 文件 / 文件夹（资源管理器里复制的文件路径）
- 自动去重：连续复制相同内容只记一条
- 隐私：自动跳过密码管理器标记的敏感内容（1Password / KeePass / Bitwarden 等）

### 2.2 卡片列表
- 打开窗口后，记录以卡片形式按**时间从新到旧**排列
- 置顶卡片始终排在最前（独立置顶区）
- 每张卡片可：
  - **点击 = 复制到系统剪贴板**（用户自己去目标处 Ctrl+V）
  - **置顶 / 取消置顶**
  - **删除**（单条）

### 2.3 搜索
- 顶部搜索框，实时筛选
- 文字按内容匹配；图片 / 文件按摘要（文件名、尺寸）匹配

### 2.4 自动清理
- 按保存天数清理；档位：1 / 3 / 5 / 7 / 30 天 / 永久（默认 **3 天**）
- 同时限制最大 **200 条**，超出删最旧的
- **置顶项不参与清理**

### 2.5 设置
- 可在软件内调整"保存天数"档位
- 最大条数固定 200（先写死，后期可开放）

## 3. 交互与启动

- 常驻**系统托盘**（右下角）：左键点击开关窗口，右键菜单可"打开 / 退出"
- 关闭窗口 = **隐藏到托盘**，不退出
- **手动启动**（不开机自启）。软件未运行时复制的内容不会被记录——窗口内有一句温和提示
- 单实例运行：再次启动只唤起已有窗口

## 4. 界面

- 风格：简洁、直观
- 主色调：**淡蓝色**（具体色值见 `03-设计规范.md`）
- 布局：顶部搜索框 + 设置入口；下方卡片列表（时间倒序，置顶区在上）；底部"未运行不记录"提示

## 5. 非功能要求

- 数据全部存本机，不上传任何服务器
- 轻量、稳定，长时间后台运行不卡

## 6. 验收口径

1. 复制文字 / 截图 / 文件 → 窗口出现 3 张对应卡片，时间倒序
2. 点卡片 → 目标处 Ctrl+V 能正确粘出
3. 置顶跳到最前；删除立即消失
4. 搜索关键字 → 只显示匹配卡片
5. 保存天数设为 1 天 → 旧记录被清；连续复制超 200 条 → 最旧的被删
6. 用密码管理器复制密码 → 不出现在列表
7. 打包 .exe → 安装后桌面有图标，双击可用
