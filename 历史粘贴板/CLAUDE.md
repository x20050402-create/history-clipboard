> 本文件是项目总入口。每次开发前请先读这里，再按"工作流程"推进。

## 项目简介

运行在 Windows 上的**历史剪贴板**软件：后台自动记录复制的**文字、图片、文件**，按时间倒序展示成卡片，可点击复制、置顶、删除、搜索；按天数 / 条数自动清理。常驻系统托盘，手动启动。

- 技术栈：Electron + better-sqlite3 + electron-builder
- 数据存放：`%APPDATA%/剪贴历史/`（`clip-history.db`、`images/`、`settings.json`），全部本机，不上传
- UI：简洁直观，淡蓝主色调

## 运行方式（在 VS Code 中体验）

本机 Node 在 `C:\Program Files\nodejs`，Git Bash 需先加 PATH：

```bash
export PATH="$PATH:/c/Program Files/nodejs"
npm install   # 首次或依赖变更时
npm start     # 运行体验
npm run dist  # 打包成 .exe（阶段 4）
```
